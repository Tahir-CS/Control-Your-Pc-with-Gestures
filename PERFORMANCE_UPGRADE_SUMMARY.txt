Problem: App still slow
→ Solution: Check if controller.exe exists in project folder

Problem: Cursor jittery

CUSTOMIZATION
====================================================================

You can adjust cursor smoothness in:
electron/windowsController.js

const SMOOTHING_FACTOR = 0.3; // Current setting

Values:
  0.1 = Very smooth, slower (good for shaky tracking)
  0.3 = Balanced (default, recommended)
  0.5 = Responsive, less smoothing
  0.9 = Very fast, minimal smoothing
